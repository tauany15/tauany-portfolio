# Tauany Portfolio

Hi, I'm Tauany. This is my personal portfolio, where I share a bit about my background, the technologies I work with, and the projects I've been building as I grow as a Software Developer.

I enjoy creating clean, accessible web experiences and approaching problems with patience, curiosity, and attention to real user needs.

Live site: [tauany15.github.io/tauany-portfolio](https://tauany15.github.io/tauany-portfolio)

## Features

- Responsive single-page portfolio layout
- Smooth navigation between Home, Projects, About, Skills, and Contact sections
- Selected project cards with GitHub and live demo links
- About section with professional background and education
- Skills section grouped by frontend, backend, database, testing, and tools
- Resume download link
- Contact links for GitHub, LinkedIn, and email

## Tech Stack

- React
- JavaScript
- Tailwind CSS
- lucide-react
- Create React App
- GitHub Pages

## Getting Started

To run this project locally:

```bash
npm install
npm start
```

The app will open at:

```text
http://localhost:3000/tauany-portfolio
```

## Available Scripts

```bash
npm start
```

Runs the app in development mode.

```bash
npm run build
```

Creates an optimized production build in the `build` folder.

```bash
npm run deploy
```

Builds and deploys the site to GitHub Pages.
